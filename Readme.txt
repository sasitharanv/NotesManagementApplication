1. Frontend Setup

Open the Frontend Folder
Navigate to the frontend folder in your terminal or command prompt:



cd path/to/frontend
Install Dependencies
Run the following command to install the necessary dependencies:

npm install


2. Backend Setup
Run Spring Boot Backend
Ensure your Spring Boot backend is up and running.

3. Start the Frontend
Start the Development Server
In the frontend folder, run the following command to start the development server:

npm start
